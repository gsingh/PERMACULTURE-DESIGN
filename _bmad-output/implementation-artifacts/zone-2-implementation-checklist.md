### 2. TREE PLANTING PREPARATION

#### 2.1 Planting Plan & Layout
- [ ] **Create detailed planting map:**
  - [ ] Map all 40-50 tree locations
  - [ ] Ensure proper spacing (avoid future crowding)
  - [ ] Consider mature canopy size

- [ ] **Spacing guidelines:**
  - [ ] Large trees (Mango, Jamun): 25-30 ft apart
  - [ ] Medium trees (Guava, Pomegranate): 15-20 ft apart
  - [ ] Small trees (Moringa, Curry Leaf): 10-15 ft apart
  - [ ] Nitrogen-fixers: Interplanted between fruit trees
- [ ] **Sun exposure consideration:**
  - [ ] Tallest trees (Mango, Jamun) on south side (don't shade others)
  - [ ] Shorter trees (Guava, Ber) on north side
  - [ ] Shade-tolerant species under larger trees

#### 2.2 Planting Pit Preparation (Month 5-6)
For each tree:
- [ ] **Dig pit:** 2 ft × 2 ft × 2 ft (60 cm × 60 cm × 60 cm)
  - [ ] Larger for big trees (Mango, Jamun): 3 ft × 3 ft × 3 ft
- [ ] **Separate topsoil and subsoil:**
  - [ ] Keep topsoil aside (nutrient-rich, first 6-12 inches)
  - [ ] Discard or use subsoil for berms
- [ ] **Amend pit soil mixture:**
  - [ ] Topsoil: 50%
  - [ ] Well-rotted compost/manure: 30%
  - [ ] Sand (if clay soil): 10-15%
  - [ ] Neem cake (pest deterrent): 100-200 grams per pit
  - [ ] Rock phosphate (if available): 50 grams per pit
  - [ ] Mix thoroughly
- [ ] **Refill pit 80% with amended mixture**
  - [ ] Leave 20% space for tree root ball
- [ ] **Create water basin around pit:**
  - [ ] Build 6-inch high berm around pit (3-4 ft diameter)
  - [ ] Creates water-holding basin for watering
- [ ] **Mulch pit area:**
  - [ ] Apply 3-4 inch mulch layer inside basin
  - [ ] Keep mulch 3-4 inches away from future trunk position

#### 2.3 Pre-Planting Care
- [ ] **Water pits deeply 1-2 days before planting**
  - [ ] Allows soil to settle
  - [ ] Tests drainage (water should drain within 24 hours)
- [ ] **Stage saplings near planting locations**
  - [ ] Keep in shade
  - [ ] Water thoroughly
  - [ ] Organize by species (avoid mix-ups)

### 3. TREE PLANTING (MONTH 6-7 - MONSOON)

#### 3.1 Planting Technique
For each tree:
- [ ] **Pre-planting preparation:**
  - [ ] Water sapling pot thoroughly (1-2 hours before planting)
  - [ ] Remove container carefully (avoid root disturbance)
  - [ ] Inspect roots (untangle if root-bound, prune damaged roots)
  - [ ] Trim any broken or diseased branches
- [ ] **Planting process:**
  - [ ] Make final hole in pit (size of root ball)
  - [ ] Place tree at correct depth:
    - [ ] Root collar at or slightly above soil level (not buried)
    - [ ] Check depth with straight stick across pit
  - [ ] Backfill with amended soil mixture
  - [ ] Firm soil gently around roots (eliminate air pockets)
  - [ ] DO NOT compact soil heavily (reduces oxygen)
- [ ] **Immediate post-planting:**
  - [ ] Water deeply (20-30 liters per tree)
  - [ ] Create proper water basin (check berm integrity)
  - [ ] Apply 4-6 inch mulch layer (keep away from trunk)
  - [ ] Install tree stake (if needed):
    - [ ] Drive stake 12 inches from trunk (upwind side)
    - [ ] Tie tree loosely with soft material (jute, cloth)
    - [ ] Allow some movement (builds trunk strength)
- [ ] **Tree identification:**
  - [ ] Install permanent plant label
  - [ ] Record: Species, variety, planting date
  - [ ] Update planting map

#### 3.2 Species-Specific Planting Notes

**Mango:**
- [ ] Choose sunny location (full sun essential)
- [ ] Plant on slight elevation (good drainage)
- [ ] Orient graft union away from prevailing wind
- [ ] Prune to 3-4 main branches after planting

**Guava:**
- [ ] Tolerates partial shade
- [ ] Can be planted closer (15 ft spacing)
- [ ] Prune to 4-5 main branches

**Jamun:**
- [ ] Very deep roots (don't plant near structures)
- [ ] Needs full sun
- [ ] Minimal pruning after planting

**Pomegranate:**
- [ ] Prefers well-drained soil (add extra sand if clay)
- [ ] Can tolerate some alkalinity
- [ ] Prune to multi-trunk form (3-5 stems from base)

**Moringa:**
- [ ] Fast-growing (harvest leaves within 3-4 months)
- [ ] Plant from seed or sapling
- [ ] Prune aggressively for bushy growth and continuous harvest

### 4. POST-PLANTING CARE (MONTHS 7-12)

#### 4.1 Watering Schedule

**Month 7-9 (Monsoon season):**
- [ ] **Week 1-2 post-planting:** Daily watering (10-15 L/tree)
  - [ ] Even during monsoon (ensures establishment)
- [ ] **Week 3-4:** Every 2-3 days (monitor soil moisture)
- [ ] **Month 2-3:** Weekly (if no rain for >5 days)
- [ ] **Adjust based on rainfall:**
  - [ ] Check soil moisture 4 inches deep (finger test)
  - [ ] If dry, water; if moist, skip

**Month 10-12 (Post-monsoon/winter):**
- [ ] **Frequency:** Twice weekly
- [ ] **Volume:** 15-20 L/tree (less frequent, deeper watering)
- [ ] **Method:** Slow, deep watering (fills basin, soaks in)
- [ ] **Timing:** Early morning or evening (avoid midday heat)

**Irrigation system:**
- [ ] Install drip irrigation (ideal for water efficiency):
  - [ ] 2-4 emitters per tree (depending on size)
  - [ ] 4 L/hour emitters
  - [ ] Run for 3-4 hours = 12-16 L per tree per session
  - [ ] Automated timer (optional but saves labor)
- [ ] OR manual watering:
  - [ ] Use hose or watering cans
  - [ ] Family rotation (different family each week)

#### 4.2 Mulching Maintenance
- [ ] **Monthly mulch replenishment:**
  - [ ] Check mulch depth (should be 4-6 inches)
  - [ ] Add more mulch as it decomposes
  - [ ] Source mulch: Crop residues, dried grass, animal bedding
- [ ] **Mulch ring expansion:**
  - [ ] Expand mulch ring as tree canopy grows
  - [ ] Target: Mulch to drip line of canopy
- [ ] **Mulch material diversity:**
  - [ ] Mix different materials (grass, leaves, straw)
  - [ ] Adds nutrient diversity as it decomposes
  - [ ] Avoid allelopathic mulches (Eucalyptus, Walnut)

#### 4.3 Weed Management
- [ ] **Hand weeding:**
  - [ ] Remove weeds within 3-foot radius of young trees
  - [ ] Compete for water and nutrients
  - [ ] Do NOT use herbicides (permaculture principle)
- [ ] **Living mulch/cover crops:**
  - [ ] Plant nitrogen-fixing cover crops between trees:
    - [ ] Cowpea, cluster bean, mung bean
  - [ ] Chop and drop as mulch when flowering
  - [ ] Seeds for next season
- [ ] **Chicken tractoring:**
  - [ ] Rotate chickens through planting areas
  - [ ] They eat weeds, seeds, insects
  - [ ] Provide fertilizer while clearing
  - [ ] Avoid during hot summer (can stress trees)

#### 4.4 Fertilization Schedule

**Month 8 (Post-monsoon):**
- [ ] Apply compost/manure around each tree
  - [ ] Quantity: 5-10 kg per tree
  - [ ] Spread in mulch ring, avoid direct trunk contact
  - [ ] Water in thoroughly

**Month 11 (Pre-winter):**
- [ ] Apply balanced organic fertilizer
  - [ ] Vermicompost: 3-5 kg per tree
  - [ ] OR well-rotted FYM: 10-15 kg per tree
  - [ ] Neem cake: 200 grams per tree (pest deterrent)

**Ongoing (Monthly):**
- [ ] Collect animal manure from chickens/ducks
- [ ] Compost for 2-3 months
- [ ] Apply as top-dressing around trees

#### 4.5 Pruning & Training

**Month 9-10 (Post-establishment):**
- [ ] **Formative pruning:**
  - [ ] Remove crossing/rubbing branches
  - [ ] Establish main scaffold branches (3-5 for most trees)
  - [ ] Remove branches <18 inches from ground (future clearance)
  - [ ] Maintain central leader (Mango, Jamun) or open vase (Guava, Pomegranate)
- [ ] **Pruning technique:**
  - [ ] Clean, sharp pruners (disinfect between trees)
  - [ ] Cut at 45° angle just above bud
  - [ ] Remove no more than 20-25% of canopy in first year
- [ ] **Stake adjustment:**
  - [ ] Loosen ties if tree growing (prevent girdling)
  - [ ] Remove stakes after 1 year (most trees)

#### 4.6 Pest & Disease Monitoring

**Weekly inspection:**
- [ ] Walk through Zone 2, observe each tree
- [ ] Check for:
  - [ ] Leaf damage (chewing, discoloration)
  - [ ] Pest presence (aphids, caterpillars, borers)
  - [ ] Fungal signs (spots, wilting, mold)
  - [ ] Trunk damage (bark splitting, gummosis)
- [ ] Document observations (photo + notes)

**Organic pest control methods:**
- [ ] **Neem spray:**
  - [ ] Neem oil (10 mL) + water (1 L) + soap (1 mL emulsifier)
  - [ ] Spray every 2 weeks (preventative) or weekly if pests present
- [ ] **Beneficial insects:**
  - [ ] Encourage predatory insects (ladybugs, lacewings)
  - [ ] Plant insect-attracting flowers (marigold, fennel)
- [ ] **Physical removal:**
  - [ ] Hand-pick large pests (caterpillars, beetles)
  - [ ] Crush or feed to chickens
- [ ] **Companion planting:**
  - [ ] Interplant with pest-deterrent species:
    - [ ] Marigold (nematodes, aphids)
    - [ ] Basil (flies, mosquitoes)
    - [ ] Lemongrass (general insect repellent)

#### 4.7 Tree Protection from Animals

**During chicken/duck rotation:**
- [ ] **Temporary tree guards:**
  - [ ] Install wire mesh cylinders around young tree trunks
  - [ ] Height: 2-3 feet (prevent pecking/scratching damage)
  - [ ] Diameter: 12-18 inches (tree can grow inside)
  - [ ] Remove after 2-3 years (tree bark toughens)
- [ ] **Timing:**
  - [ ] Install BEFORE introducing animals to paddock
  - [ ] Chickens/ducks attracted to tender bark of young trees
- [ ] **Alternative:**
  - [ ] Delay animal rotation in newly planted areas
  - [ ] Wait 1 year for trees to establish before animal integration

## TROUBLESHOOTING GUIDE

### Common Challenges & Solutions

**Problem: Low egg production**
- Causes: Poor nutrition, stress, age, disease, inadequate light
- Solutions:
  - [ ] Switch to higher quality layer feed (18% protein)
  - [ ] Supplement with calcium (oyster shell grit)
  - [ ] Reduce stressors (predators, overcrowding, heat)
  - [ ] Cull old/low-producing birds, replace with young pullets
  - [ ] Provide 14-16 hours light (artificial light in winter if needed)

**Problem: High animal mortality**
- Causes: Predation, disease, poor nutrition, heat stress
- Solutions:
  - [ ] Strengthen fencing/coop security (bury wire, lock at night)
  - [ ] Vaccinate birds (Ranikhet, fowl pox)
  - [ ] Improve ventilation (reduce ammonia buildup)
  - [ ] Provide shade and cooling (summer)
  - [ ] Quarantine new birds (prevent disease introduction)

**Problem: Slow tree growth**
- Causes: Poor soil, inadequate water, pest damage, competition
- Solutions:
  - [ ] Increase compost application (10-15 kg/tree)
  - [ ] Improve irrigation (drip system, deep watering)
  - [ ] Control pests organically (neem spray, beneficial insects)
  - [ ] Reduce weed competition (mulching, hand weeding)
  - [ ] Test soil, amend deficiencies (N, P, K, micronutrients)

**Problem: Predator attacks**
- Causes: Weak fencing, nighttime exposure, attracting predators (food left out)
- Solutions:
  - [ ] Reinforce fencing (bury wire, electric fence)
  - [ ] ALWAYS close coop/house at night (no exceptions)
  - [ ] Remove feed at night (attracts rodents, then predators)
  - [ ] Motion-activated lights (deter nocturnal predators)
  - [ ] Guard animals (dog, if feasible - advanced)

**Problem: Soil still poor after 1-2 years**
- Causes: Insufficient organic matter, inadequate animal integration, compaction
- Solutions:
  - [ ] Increase animal rotation frequency (more manure input)
  - [ ] Add green manure cover crops (legumes, chop-and-drop)
  - [ ] Import compost if needed (short-term solution)
  - [ ] Plant more nitrogen-fixers (Subabul, Gliricidia)
  - [ ] Avoid compaction (limit heavy equipment, mulch pathways)
