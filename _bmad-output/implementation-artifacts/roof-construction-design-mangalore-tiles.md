### Tile Installation Method

**1. Batten Preparation:**

**Batten Spacing:**
- Tile length: 14 inches (average)
- Tile overlap: 3-4 inches (minimum 3 inches for monsoon)
- Exposed length: 10-11 inches
- **Batten spacing: 10 inches on center** (up the slope)

**Batten Leveling:**
- Use string line from eave to ridge
- Check level with spirit level every 10 feet
- **Critical:** First batten at eave must be perfectly level (sets alignment for all tiles)

**Batten Attachment:**
- Nail to purlins: 3-inch galvanized nails, 2 nails per purlin intersection
- Spacing: Batten nails 12-16 inches apart along batten length

---

**2. Tile Installation (Bottom to Top):**

**Starting Row (Eave):**
- Install first row of tiles at eave
- Tiles overhang batten by 2-3 inches (drip edge)
- Secure every 3rd or 4th tile with copper wire or nail (others held by gravity + overlap)
- **Pattern:** Alternate "pan" (trough) and "cover" (cap) tiles
  - Pan tile: Sits on batten, curved side down (collects water)
  - Cover tile: Overlaps edge of pan tiles, curved side up (sheds water)

**Subsequent Rows:**
- Each row overlaps previous by 3-4 inches
- Offset tiles by half width (staggered pattern - prevents water infiltration)
- Secure perimeter tiles and ridge tiles (high wind areas)

**Ridge Cap:**
- Special ridge tiles OR cut tiles to fit ridge angle
- Mortar ridge tiles in place (cement-sand 1:4)
- Install ridge vent under ridge tiles (see below)

**Valley Installation (if applicable):**
- Metal valley flashing under tiles (galvanized steel or copper)
- Width: 18-24 inches (12 inches up each side of valley)
- Tiles cut to fit valley edges, mortared in place

---

**3. Tile Fastening (Wind Resistance):**

**Minimum Fastening (Low Wind Area):**
- Secure every 5th tile row
- Secure all perimeter tiles (eave, rake, ridge)
- Method: Copper wire through tile holes, tied to batten

**Standard Fastening (Kurukshetra - Moderate Wind):**
- Secure every 3rd tile row
- Secure perimeter + every 4th tile along row

**Hurricane/High Wind Fastening (Optional):**

## ðŸ—ï¸ ROOF CONSTRUCTION PROCESS (Step-by-Step)

### Phase 1: Structural Frame (Week 1-2)

**Step 1: Ridge Beam Installation (Day 1-2)**

**Process:**
1. **Prepare gable walls:**
   - Top of gable walls must be level (check with spirit level)
   - Mark ridge beam position (centered on wall, correct height)
   - If using timber: Create notch or install beam pocket in wall
   - If using steel: Weld/bolt steel plate to wall top, beam sits on plate

2. **Lift ridge beam into position:**
   - Timber beam: 6"Ã—8"Ã—30 ft weighs ~400 lbs (180 kg) - requires 4-6 workers OR crane
   - Use scaffold or ladder at both gable ends
   - Lift beam, rest on gable walls
   - Check level (beam must be perfectly level along length)
   - Secure beam (anchor to wall with bolts or steel straps)

**Safety:** Ridge beam is heaviest single piece - use proper lifting technique or machinery

---

**Step 2: Rafter Installation (Day 3-7)**

**Process:**
1. **Mark rafter positions:**
   - Chalk lines on ridge beam: 18 inches on center (both sides of beam)
   - Chalk lines on wall plates: Match ridge beam spacing

2. **Cut rafters:**
   - Length: 20.5 ft (includes 3 ft overhang at eave)
   - Ridge cut: 30Â° angle (fits against ridge beam)
   - Eave cut: 60Â° plumb cut (vertical cut at eave end)
   - Bird's mouth: Notch where rafter sits on wall plate (4 inches deep Ã— 6 inches wide)

3. **Install rafters:**
   - Start at gable ends (outside rafters)
   - Position rafter: Ridge end against beam, bird's mouth on wall plate
   - Nail to ridge beam: 3-inch galvanized nails, 3 nails per connection (toe-nailing)
   - Nail to wall plate: 3 nails through bird's mouth
   - Install remaining rafters: Work from one gable to other, maintain 18" spacing
   - Check plumb: Use level on each rafter (must be vertical when viewed from end)

4. **Install collar ties (if required):**
   - Collar ties: Horizontal members connecting rafters (2/3 up from wall plate)
   - Purpose: Prevent rafter spread (outward thrust)
   - Size: 2"Ã—6" boards
   - Spacing: Every 4 feet (every 3rd rafter pair)
   - Install if: Span > 20 ft OR roof pitch < 4:12 (your 30Â° = 7:12 is OK, but adding them is good practice)

**Timeline:** 5 days for 38 rafters (2 sides Ã— 19 rafters) + collar ties

---

**Step 3: Purlin Installation (Day 8-10)**

**Process:**
1. **Mark purlin positions:**
   - Spacing: 24 inches up the slope (from eave to ridge)
   - Snap chalk lines across all rafters (perpendicular to rafters)

2. **Cut purlins:**
   - Length: 30 ft (full house length + overhangs)
   - No angle cuts (purlins are straight)

3. **Install purlins:**
   - Lay purlin across rafters (on chalk line)
   - Nail to each rafter: 3-inch galvanized nails, 2 nails per rafter
   - Continue up slope (9 purlins per side)
   - Check alignment: String line from end to end (purlins must be straight)

**Timeline:** 3 days for 18 purlins (2 sides Ã— 9 purlins)

---

### Phase 2: Roof Deck & Insulation (Week 3)

**Step 4: Install Underlayment (Day 1-2)**

**Process:**
1. **Start at eave:**
   - Roll out synthetic underlayment horizontally (parallel to eave)
   - Overlap eave by 6 inches (drip edge)
   - Staple to rafters: Every 12 inches along edges, 24 inches in field

2. **Subsequent rows:**
   - Overlap previous row by 4-6 inches (manufacturer spec)
   - Stagger seams (don't align all seams vertically)
   - Continue to ridge

3. **Ridge treatment:**
   - Overlap underlayment from both sides at ridge (minimum 6 inches)
   - Install ridge vent opening AFTER underlayment (cut underlayment as needed)

**Timeline:** 2 days for 106 sq m underlayment

---

**Step 5: Install Battens (Day 3-4)**

**Process:**
1. **Mark batten spacing:**
   - Measure 10 inches up from eave (first batten)
   - Snap chalk line across rafters
   - Continue every 10 inches up to ridge

2. **Cut battens:**
   - Length: 30 ft (full house length + overhangs)
   - No angle cuts

3. **Install battens:**
   - Nail to purlins: 3-inch galvanized nails, 2 nails per purlin intersection
   - Battens should be straight and level (check with string line)
   - Install 21 batten rows per side (north and south)

**Timeline:** 2 days for 42 batten rows (2 sides Ã— 21 rows)

---

**Step 6: Install Insulation (Day 5-7)**

**Process (using straw bales):**
1. **Prepare straw:**
   - Spray with borax fire retardant (1 kg : 5 L water)
   - Let dry 24-48 hours before installing

2. **Install radiant barrier (if using):**
   - Staple aluminum foil/bubble wrap to underside of rafters
   - Leave 1-inch air gap between barrier and roof deck

3. **Lay straw bales:**
   - Between rafters (on top of ceiling joists, if ceiling present)
   - OR over rafters (if no ceiling, lay on flat surface below)
   - Compress bales to fit tightly
   - Wire bales together (prevent shifting)

4. **Cover straw:**
   - Lay breathable fabric over straw (burlap or cotton canvas)
   - Prevents straw from shedding into living space
   - Allows moisture vapor to pass

**Timeline:** 3 days for insulation installation

---

### Phase 3: Tile Installation (Week 4-5)

**Step 7: Install Tiles (Day 1-10)**

**Process:**
1. **Start at eave:**
   - Lay first row of pan tiles (trough side down, on first batten)
   - Overhang eave by 2-3 inches (drip edge)
   - Space tiles 1-2 inches apart (side-to-side)

2. **Install cover tiles (first row):**
   - Lay cover tiles over pan tile edges (cap the joints)
   - Overlap pan tiles by 2-3 inches

3. **Second row:**
   - Lay pan tiles (overlap first row by 3-4 inches - expose 10 inches of first row)
   - Offset tiles by half width (staggered pattern)
   - Continue pattern up to ridge

4. **Secure tiles:**
   - Wire every 3rd row (copper wire through tile holes, twisted around batten)
   - Secure all perimeter tiles (eave, rake, ridge)

5. **Ridge tiles:**
   - Install ridge vent cover first
   - Mortar ridge tiles over vent (cement-sand 1:4)
   - Overlap ridge tiles 3-4 inches
   - Point mortar joints (fill gaps between tiles)

**Timeline:** 10 days for ~1,200 tiles per house (2 workers, 120 tiles/day)

---

### Phase 4: Finishing (Week 6)

**Step 8: Install Ridge Vent Cover (Day 1)**
- Already installed under ridge tiles (Step 7)

**Step 9: Install Soffit Vents (Day 2)**
- Cut soffit vent openings
- Install vent covers
- Check airflow (feel air movement from soffit to ridge)

**Step 10: Install Gutters & Downspouts (Day 3-4)**
- Attach gutter brackets to fascia board (every 2-3 ft)
- Hang gutters (slope toward downspouts)
- Install downspouts at corners
- Connect downspouts to drainage (splash blocks or underground pipes)

**Step 11: Final Inspection (Day 5)**
- Check tile alignment (any missing or cracked tiles?)
- Check flashing (valleys, penetrations sealed?)
- Check gutters (water flows to downspouts?)
- Check ventilation (soffit to ridge airflow?)
- Water test (hose on roof, check for leaks inside)

**Timeline:** 5 days for finishing work

- Sequential (one at a time): 24 weeks (6 months)
