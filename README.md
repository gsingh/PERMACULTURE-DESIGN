# PERMACULTURE FARM PROJECT - KURUKSHETRA, HARYANA

## Project Overview
A comprehensive permaculture design for establishing a sustainable farm in Kurukshetra, Haryana. This project aims to create a resilient, productive, and ecologically harmonious agricultural system that works with nature rather than against it.

## Location
- **Location**: Kurukshetra, Haryana, India
- **Climate**: Semi-arid with hot summers, mild winters, and seasonal rainfall
- **Soil**: Alluvial plains with fertile soil
- **Rainfall**: Moderate seasonal rainfall (monsoon-dependent)
